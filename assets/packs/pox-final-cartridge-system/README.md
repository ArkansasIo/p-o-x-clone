# POX Final Cartridge System

This folder contains all scripts, tools, and assets for building, packing, and exporting PO-X cartridges.

## Included Tools
- ROM builder script
- Cartridge packer
- Save-slot builder
- Procedural sprite generator
- Boot animation assets
- Documentation

## Structure
- scripts/: Python and batch scripts for ROM and asset building
- assets/: Boot animation, sprites, and cartridge templates
- docs/: Guides and API documentation
