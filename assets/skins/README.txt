# Skins Asset Template

To add a new skin for a creature, create a PNG file named after the creature (e.g., hero_skin.png, monster_skin.png) and place it in this folder.

Recommended size: 64x64 pixels (or as needed).

Example filenames:
- hero_skin.png
- monster_skin.png
- dragon_skin.png
- goblin_skin.png
- elf_skin.png
- ...

---
For handheld skins, use the naming convention: hero_handheld_skin.png, monster_handheld_skin.png, etc.
