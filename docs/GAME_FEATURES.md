# PO-X Clone Game Features

This document summarizes the main features of the PO-X clone game and emulator platform.

## Core Features
- Digital pet/creature system (20-byte save record, CRC, SRAM slots)
- Menu UI (Feed, Play, Heal, Save, Load)
- Input handling (D-pad, A/B/Start/Select)
- VRAM-based display pipeline (2bpp, 4-shade palette)
- Save/load system (SRAM, .sav files)
- Debugging tools (trace, breakpoints, watchpoints)
- ROM container format (POXR)
- Homebrew SDK and example projects


## Usage

Refer to this document to understand the current and planned capabilities of the PO-X Clone platform. Use the features listed above to guide development, testing, and gameplay.
