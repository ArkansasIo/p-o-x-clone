# POX Game Design Document (GDD)

## 1. Game Overview
- Explore 999+ procedural mazes, dungeons, towers, and worlds.
- Capture, evolve, and equip over 600 body parts, weapons, and armor.

## 2. Core Systems
- Maze/trial/raid/tower/world generator
- RPG/MMO: character, party, quest, inventory, combat
- Game database: creatures, items, parts, etc.
- Universe/room: procedural planets, multiplayer
- Resource/tech: gathering, tech tree, upgrades
- Wireless: unified multiplayer API
- Config: unified settings

## 3. Key Features
- Procedural content, persistent universe
- PvP, PvE, raids, world events
- Trading, guilds, chat, mail
- Save/load, firmware, patch, EEPROM
