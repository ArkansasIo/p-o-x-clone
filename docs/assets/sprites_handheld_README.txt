# Handheld Sprite Asset Template

To add a new handheld creature sprite, create a PNG file named after the creature (e.g., hero_handheld.png, monster_handheld.png) and place it in this folder.

Recommended size: 32x32 or 64x64 pixels (for handheld display).

Example filenames:
- hero_handheld.png
- monster_handheld.png
- dragon_handheld.png
- goblin_handheld.png
- elf_handheld.png
- ...

---
See handheld_creature_list.txt for all required creatures.
