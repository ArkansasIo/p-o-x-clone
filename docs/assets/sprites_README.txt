# Sprite Asset Template

To add a new creature sprite, create a PNG file named after the creature (e.g., hero.png, monster.png) and place it in this folder.

Recommended size: 64x64 pixels (or as needed for handheld display).

Example filenames:
- hero.png
- monster.png
- dragon.png
- goblin.png
- elf.png
- ...

---
For handheld sprites, use the naming convention: hero_handheld.png, monster_handheld.png, etc.
